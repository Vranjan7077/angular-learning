How It Works

This demo uses runtime translation with ngx-translate.

Flow

  1. Select a language from the dropdown.
  2. The app loads the matching messages.<lang>.json file.
  3. The translate pipe updates the UI text.
  4. The {{ placeholderExample }} placeholder is replaced with your input.
